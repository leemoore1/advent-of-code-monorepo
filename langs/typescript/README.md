# Advent of Code 2024

My source code for Advent of Code 2024 puzzle work, in TypeScript.

## Requirements

- Node.js: v22.4.0
- pnpm: 9.12.0

## Installation

### Dependencies

```shell
pnpm install
```

### Puzzle inputs

Save puzzle inputs as `.txt` files in the [inputs](./inputs) directory.

## Getting started

```shell
pnpm test
```

...will run Jest tests for all puzzles.

## Desired features

- Directions for including puzzle samples for running tests.
- pnpm script per puzzle part, e.g. `pnpm run d1:p1`
